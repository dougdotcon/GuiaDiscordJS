# Bem-vindo

## Introdução

Este guia faz o melhor esforço para humanizar o uso da biblioteca Discord.js, em sua versão atual 13. Ele é mantido por York#0001 e Discordaholic#0001 ("Evie.Codes"), com a ajuda de muitos colaboradores preciosos que nos permitem continuar atualizando e aprimorando este guia.

Ele nasceu da necessidade – a primeira versão deste guia para o Discord.js versão 7, há mais de 5 anos, foi a primeira do tipo porque eu (Evie) não conseguia encontrar nenhum guia para iniciantes e a documentação era (e ainda é) incrivelmente difícil de navegar para quem não possui conhecimento suficientemente avançado dos conceitos de JavaScript.

Neste guia, fazemos todo o possível para _ensinar_ você a usar a biblioteca e como estendê-la com seu próprio código. No entanto, esperamos que você tenha uma compreensão básica de JavaScript antes de tentar criar bots. Discord.js (e outras bibliotecas para Discord) exigem alguns conceitos intermediários a avançados que podem ser difíceis de entender para quem não tem uma boa base na linguagem.

## Obtenha Suporte

Se você tiver alguma dúvida após ler este guia, não hesite em se juntar ao nosso servidor Discord, "An Idiot's Guide". O link está localizado no topo desta página. Se desejar consultar os desenvolvedores do discord.js e a comunidade oficial, o Discord oficial do Discord.js está logo ao lado do nosso, também no topo da página!
